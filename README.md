# GeoCards

**GeoCards** фронтенд-проект для изучения географии: страны, факты, презентации с уроков, справочник терминов и квиз на знание флагов. Все это оформлено в едином фирменном стиле с анимациями и плавными переходами. Начиналось как простая идея с карточками, а выросло в небольшую интерактивную площадку для практики JavaScript, работы с API и UI-анимаций.

Демо: https://gilazovdev.github.io/geocards/

---

## Возможности

### Страны
Поиск по алфавитному справочнику стран. При клике на страну открывается карточка с данными, которые подтягиваются в реальном времени из [REST Countries](https://restcountries.com/) — столица, регион, население, площадь, языки. Дополнительно показывается текущая погода в столице страны, полученная через API [Open-Meteo](https://open-meteo.com/) по координатам, которые возвращает REST Countries.

### Презентации
Презентации больше не захардкожены в разметке они загружаются через имитацию асинхронного источника данных (со скелетон-заглушками во время загрузки), после чего их можно искать и фильтровать по теме и классу. При клике на карточку открывается мини-просмотрщик слайдов с навигацией вперёд/назад.

### Справочник
24 географических термина, разбитых по категориям (рельеф, вода, климат, геология, координаты), с аккордеоном, живым поиском и блоком «термин дня», который меняется автоматически.

### Факты
Фильтруемая подборка фактов о планете, генератор случайного факта (из "бд").

### Квиз
Квиз из 10 вопросов на знание флагов в режиме «флаг - страна» ведется счет по игре, лучшая серия правильных ответов подряд и личный рекорд. За идеальный результат (10 из 10) запускается анимация конфетти.

### Прочее
- Переключатель светлой и тёмной темы на CSS-переменных
- Анимации появления блоков при скролле
- Волновой эффект и блики при наведении на кнопки, карточки и фильтры
- Всплывающие уведомления, кнопка «наверх», прижатый к низу футер
- Полная адаптивность вплоть до небольших мобильных экранов

## Стек технологий

- **HTML5 / CSS3** — собственная дизайн-система на CSS-переменных (без CSS-фреймворков, кроме сетки и утилит Bootstrap)
- **Vanilla JavaScript** — без сборщиков и фреймворков, всё работает прямо в браузере
- **Bootstrap 5** — сетка, модальные окна, collapse-меню (подключены через CDN)
- **flag-icons** — спрайты флагов стран (через CDN)
- **REST Countries API** — данные о странах
- **Open-Meteo API** — данные о погоде в реальном времени, без API-ключа
- **localStorage** — избранные страны, лайки фактов, рекорды квиза, выбранная тема

## Планы на будущее

Проект остаётся учебным и портфолио-ориентированным, и в нем еще есть куда расти:

- Полноценный бэкенд (Node/Express или аналог) вместо клиентских данных и localStorage
- Аккаунты пользователей, чтобы результаты квиза сохранялись между устройствами
- Больше режимов квиза (сопоставление по регионам, ранжирование по площади/населению, режим на время)
- CMS или админ-панель для управления презентациями
- Автоматические тесты и полноценный пайплайн сборки
